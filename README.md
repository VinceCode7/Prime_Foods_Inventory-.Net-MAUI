# Prime_Foods_Inventory
An Inventory software to manage the products operations
- The project uses .NET MAUI framework.
- The software is able to connect to a sql server(MSSQL).
- The files used to create database tables, have been included in the Project.

  ## FEATURES
  1. Database access
  2. Nice and Simple UI and layouts for FrontEnd
  3. Simple C# backEnd code
  4. Use of POP UP Pages
  5. Error handling
  6. etc
